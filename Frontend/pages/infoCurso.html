Curso completo de programacion con JavaScript

Programa desde cero, domina Javascript, entiende HTML y aprende de algoritmos. Sí, desde cero. Entenderás la lógica del código, cómo piensan las programadoras expertas y cómo programar un videojuego web de principio a fin. Aprender a programar no es fácil, pero Acadevmia lo hace efectivo.

Temario y recursos del Curso

BASICO

15 HORAS DE CONTENIDO

14 HORAS PRACTICAS

MODULO 1

Se comienza con una visión general de qué es JavaScript, su historia y su propósito en el desarrollo web. Luego, aprenderían a configurar su entorno de desarrollo, incluyendo la elección de un editor de texto y cómo utilizar la consola del navegador para ejecutar código JavaScript. Se abordarían conceptos básicos de sintaxis, como variables, tipos de datos, operadores y declaraciones, junto con la comprensión de las estructuras de control de flujo, como condicionales (if, else) y bucles (for, while).

MODULO 2

Los estudiantes se adentrarían en la manipulación del DOM (Modelo de Objetos de Documento). Comprenderían la relación entre el DOM y JavaScript, aprendiendo a seleccionar elementos HTML en una página web utilizando JavaScript. Se enseñaría cómo manipular el contenido de estos elementos, como cambiar texto, modificar atributos y estilos, y cómo responder a eventos del usuario mediante la adición de "event listeners".

MODULO 3

Comenzarían con los fundamentos de la POO y cómo se aplican en JavaScript. Se enseñaría la creación de objetos y propiedades, así como el uso de constructores y prototipos para definir clases y métodos. Los estudiantes explorarían conceptos avanzados como la herencia y la encapsulación en JavaScript, y cómo utilizarlos para organizar y extender el código de manera efectiva.
